# 📈 QuantCrypt

QuantCrypt is a modular crypto trading bot with backtesting capabilities.

---

## 🚀 Features

- SMA + RSI trading strategy
- Backtesting engine
- Binance API support via CCXT
- Modular architecture
- Risk management configuration

---

```bash
git clone https://github.com/doomshadowx/quantcrypt.git
cd quantcrypt
pip install -r requirements.txt
